A simple, educational game for toddlers created using Python and Pygame. Toddlers can press any key or click the mouse to change the displayed image and hear a corresponding sound.

## Requirements

- Python 3.x
- Pygame

## Installation

1. Clone this repository or download it as a zip file.
```bash
git clone https://github.com/zediknight/ToddlerTapGame.git
```

2.  Install Python 3.x from [https://www.python.org/downloads/](https://www.python.org/downloads/).
3.  Install Pygame by opening a terminal or command prompt and typing:
```bash 
pip install pygame
```

## Usage

1.  Place your image files (in .png, .jpg, .jpeg, .gif, or .bmp format) in the "images" folder.
2.  Place your sound files (in .wav or .mp3 format) in the "sounds" folder.
3.  Run the game by opening a terminal or command prompt, navigating to the project directory, and typing:
```bash 
python main.py
```

## How to Play

-   Press any key or click the mouse to change the displayed image and hear a corresponding sound.
-   To close the game, click the close button on the window.
